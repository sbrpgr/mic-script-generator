    개인정보처리방침 이용약관

    Service Principles

    서비스 원칙

    1. 즉시 실행 회원가입이나 로그인 없이 곧바로 도구를 열고 결과를 만들 수 있습니다.
    2. 브라우저 처리 초기 도구는 서버 API 없이 브라우저 안에서 텍스트와 파일을 직접 처리합니다.
    3. 보안 우선 민감한 파일과 텍스트는 업로드하지 않고, 배포용 토큰은 클라이언트 코드에 넣지 않습니다.
    4. 광고 분리 작업 UI와 광고 슬롯을 분리해 나중에 AdSense를 넣어도 사용 흐름이 깨지지 않도록 구성합니다.
